# Evaluation Metrics

The evaluator supports ranking, validity, and pre-experiment beyond-accuracy metric helpers.

Required families:

- Recall@K, HitRate@K, NDCG@K, MRR@K;
- validity and hallucination rate;
- catalog coverage and item coverage;
- intra-list diversity;
- novelty;
- long-tail ratio and popularity buckets;
- domain, user-sparsity, and item-popularity segments;
- latency, token, cost, and throughput summaries.

Paper tables must be exported from metrics artifacts, not manually typed.
